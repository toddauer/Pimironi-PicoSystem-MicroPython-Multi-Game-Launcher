# Pimironi-PicoSystem-MicroPython-Multi-Game-Launcher
Pimironi PicoSystem MicroPython vibe-coded game development and multi-game launcher. Want to easily create sprite and non-sprite based games in MicroPython for this system? I have a method.


Not everything works, but it is pretty fun.

Make sure you get the pimoroni-picosystem-v1.0.0-micropython.uf2 from https://github.com/pimoroni/picosystem/tree/main/micropython#get-the-latest-release

From Thonny, dump the files exactly as they are into your PicoSystem memory.
